# Weather App – DevOps Project

## Project Description
This project is a Weather Web Application built using HTML, CSS, and JavaScript.  
The application fetches real-time weather data and displays temperature, humidity, and wind speed.

This project is also used to demonstrate a complete DevOps pipeline using GitHub, Jenkins, and Docker.


## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### DevOps Tools
- Git & GitHub (Version Control)
- Jenkins (CI/CD Pipeline)
- Docker (Containerization)
- Nginx (Web Server inside Docker)

## Features
✔ Search weather for any city  
✔ Shows temperature  
✔ Shows wind speed  
✔ Shows humidity  
✔ Responsive UI  
✔ No API Key version available  

## Flow
GitHub → Jenkins → Docker Build → Container Run → Deploy App
